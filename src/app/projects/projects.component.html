Master thesis

I wrote my master's thesis "Developing a Scalable and Adaptive Cybersecurity Serious Game: A Modular 2D RPG Framework" at the Chair of Human-Centred Security. It is about the development of a 2D RPG to strengthen IT security awareness. The main focus is on the implementation. The software is to be designed in such a way that it can be extended and modified for future (empirical) research purposes. Finally, a pilot study will be carried out.

The game's backend is developed using Node.js, and the WebGL build of the game is hosted on Netlify. The database used for the project is MongoDB.

The game will be available after the study has been conducted.
